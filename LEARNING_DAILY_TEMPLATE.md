# Daily Learning Output Template

Use this as your daily proof-of-learning artifact for the 14-day plan.

## How to Use

1. Copy this file to `learning_outputs/day-XX.md` (example: `learning_outputs/day-03.md`).
2. Fill only the fields for the current day.
3. Keep entries concise and evidence-based (commands, outputs, file paths).
